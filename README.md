# DentApp

DentApp — современный стоматологический онлайн-сервис, позволяющий пациентам записываться на приём, проходить предварительную диагностику и управлять историей посещений. Для врачей предусмотрен интерфейс управления расписанием.

## Основные функции

- **Аутентификация** — регистрация и вход в личный кабинет (данные сохраняются в `localStorage`).
- **Запись на приём** — пошаговый выбор услуги, врача, даты и времени.
- **Предварительная диагностика** — опрос о симптомах с формированием рекомендаций.
- **Личный кабинет** — просмотр ближайших приёмов, статистики посещений и быстрые действия.
- **История посещений** — фильтрация и детальный просмотр прошлых визитов.
- **Расписание для врачей** — календарь с доступными и занятыми слотами (демо-режим).

## Технологии

- HTML5
- CSS3 (адаптивная вёрстка)
- JavaScript (vanilla)
- Font Awesome для иконок
- Хранение данных — `localStorage` браузера

## Структура файлов

- `DentApp.html` — основная разметка страницы.
- `style.css` — все стили.
- `script.js` — логика приложения: навигация, обработка форм, работа с `localStorage`, моковые данные.

## Запуск проекта

1. Скачайте или склонируйте репозиторий.
2. Откройте файл `DentApp.html` в любом современном браузере.
3. Для полноценной работы рекомендуется использовать локальный сервер (например, Live Server в VS Code), но приложение работает и просто при открытии файла.

## Использование

- **Гость** может просматривать главную страницу и возможности сервиса.
- **Зарегистрированный пользователь** получает доступ к личному кабинету, записи на приём, диагностике и истории.
- Для демонстрации используются моковые врачи, услуги и вопросы диагностики.
- Данные о пользователях, записях и истории хранятся в `localStorage`. При первом входе можно создать тестовый аккаунт.

## Особенности

- Все страницы переключаются без перезагрузки (SPA-подход).
- Формы проходят базовую валидацию.
- Интерфейс адаптирован под мобильные устройства.
- Для врачей доступна вкладка «Расписание» (имитация управления слотами).
